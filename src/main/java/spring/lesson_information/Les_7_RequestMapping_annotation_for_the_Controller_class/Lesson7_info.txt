*** Les_7_Аннотация @RequestMapping для Controller класса ***

На этом уроке мы разберем, что аннотацию @RequestMapping, которую мы использовали для методов внутри
контроллера, можно использовать не только для методов, но и для самого класса MyController тоже.

Адрес "/employee" который мы указываем в @RequestMapping("/employee") над классом, будет добавляться
в начало адреса каждого метода. Смотреть слайд №1.

@RequestMapping который мы пишем над классом называется Controller Mapping.
@RequestMapping который мы пишем над методом называется Method Mapping.
