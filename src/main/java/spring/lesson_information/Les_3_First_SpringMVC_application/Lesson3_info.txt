*** Les_3 *** Первое Spring MVC приложение.

На этом уроке мы создадим контроллер, которому DispatcherServlet (Front Controller) будет
перенаправлять http request. И создадим простейший View, который будет отображаться, когда мы в браузере
будем набирать конкретный адрес. Компонент модель пока затрагивать не будем, добавим его через
пару уроков.

В пакете view создадим файл first-view.jsp.
В пакете mvc создадим контроллер (Java class - MyController) с помощью которого
и будет отображаться наш view - first-view.jsp.
Далее над нашим классом мы поставим аннотацию @Controller - это специальный компонент с помощью, которого
наш контроллер будет найден при сканировании Spring-ом нашего пакета.
После того как мы закончим создание класса MyController и метода, запускаем наш Tomcat 9.0.67 и должна
открыться страница в браузере. В браузере после адреса http://localhost:8080 будет отображаться и другая информация,
это название нашего пакета и war_exploded - http://localhost:8080/spring_for_beginners_course_mvc_war_exploded/.
Что бы убрать эту ненужную информацию это можно сделать в настройках Intellij Idea.
